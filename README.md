# 基金收支记录工具

## 使用

- npm run i
- npm run serve

## 数据

暂时不打算把数据放在数据库里，所以就使用本地的数据，由 node 去更新本地的文件。

暂时两个数据，路径为：

- 基金信息：serve/data/funds.js
- 订单信息：serve/data/buys.js

> 在执行 npm run serve 的时候，会判断是否有这两个文件，无则新建
